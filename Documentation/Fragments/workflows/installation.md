## Installation

### Prerequisites

- macOS 14.0+ or Linux with Swift 6.0+
- Swift Package Manager
- Git

### From Source

```bash
git clone https://github.com/yourusername/iFoundation.git
cd iFoundation
swift build -c release
sudo cp .build/release/iFoundation /usr/local/bin/ifoundation
```

### Verify Installation

```bash
ifoundation --version
ifoundation doctor
```
